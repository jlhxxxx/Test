# 基础

## 关注点

参考：[性能测试基础（一）性能测试分类](http://www.testclass.net/performance/test_class/)

- **爆发力**就是系统能承受的最大压力，没准系统承受的压力很大，但过半个小时之间就挂掉了。
- **耐力**就是这个系统长时间处于压力下的稳定性，这系统超级稳定，跑个几十年都不用重启服务器。
- **整体协调性**就是看系统有没系统瓶颈（短板），并对其进行系统调优。
